Rules

Verification Rules

  • Videos must be permanent.
    • For Solo and Duo runs, ALL players must have full recording.
    • For Trio and Squad runs, a single runner may be allowed exemption from recording IF they are unable to, this player CANNOT be the host of the run. The provided reasoning for no video must be strong.
  • Video must start at the main menu with the version number visible.
  • Video must have in game audio.
    • Recording the player's microphone audio is strongly preferred, but not required.
    • Recording Discord audio is up to the player's discretion.
  • If playing with allowed mods, the session code must be visible at all times.
  • ALL player logs files must be saved from each session and submitted if requested by a verifier.
  • It is recommended you take no more than 4 sessions to complete a run. Runs that exceed the 4 session limit are less likely to be accepted.
    • A session starts when the host player launches the game and ends when that same player closes the game.

Gameplay Rules

  • Closing the lobby to prevent saving (aka Save Scumming) is NOT allowed.
  • Getting information about the current day from outside your point of view is NOT allowed.
  • Talking to team mates outside of the game is NOT allowed.
  • When in orbit, restarting the lobby is reserved only for when the game is currently unplayable or a player has disconnected.
  • Copying a preset save file IS allowed, so long as it doesn't have any days played.
  • Clipping out of bounds IS allowed ONLY to reach ortherwise unreachable items or to teleport back to Main Entrance.

Banned Exploits

  • Time freeze
    • Holding the game window
  • Speed glitch
  • Negative weight
  • Weather rerolling
  • Multi-terminal
    • Multiple players using the terminal
  • Cruiser vaulting
    • Keeping the items on the cruiser to avoid losing them when you wipe
  • Gift box manipulation

Other Information

  • A "session" is defined as when the save is closed. However, a session is NOT used when closing the save to reset due to a bug or disconnect.
  • If any disconnects happen during the day, the remaining players must finish out the day before resetting. If host crashes, the day can be reset & replayed.
  • Exceptions and accommodations can be made if needed, feel free to ask the moderators any questions you have.
  • If one of your runs have been verified but cannot be viewed on the leaderboard, make a ticket and we'll fix it.
  • When submitting to the board, runs can take anywhere from 1-5 weeks to be verified. Please be patient!

Classic High Quota Rules

All main rules apply.

Single Moon High Quota Rules

All main rules apply with the following additions or exceptions:

  • You must stay on the same moon all throughout the run.

Single Day Clear Rules

All main rules apply with the following additions or exceptions:

  • If running specifically for Single Day Clears, video proof must start from the main menu with the version number visible.
  • Video may only end after the final "Scrap Collected" screen has fully displayed.
  • Player logs are only required from the session the SDC occurred.
  • You may quit your save at any time to avoid losing gear.

Be respectful.

Be respectful to yourself and others, interactions should not be abusive in any way nor create an unwelcoming/toxic environment. Please don’t make tier lists/rankings of other community members or encourage any similar behavior that could put down others.

Do not use offensive language when interacting with others.

Any personal attacks or harassment will not be tolerated. Any forms of bigotry, ableism, or any other form of discrimination will not be tolerated. Any sexual comments, especially from or directed towards minors, will not be tolerated.

Breaking any of these rules will result in warnings, mutes, or bans, following a three-strike system.

Your first two infractions will be met with formal warnings. A third occurrence will be met by a temporary or permanent ban. You may continue submitting runs to the leaderboards, so long as you follow our rules.
